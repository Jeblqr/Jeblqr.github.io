#OneCode天梯赛 Round3

第一题
困难的编码

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10132

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

OneCode收到了一份奇怪的编码，据说这是先知的一份笔记，这个笔记让人难以理解，不过聪明的CXK很快就发现了这个编码的规律。
这个规律是，这个编码的总长度是两个数的乘积，并且把这个编码放在一个N*M的格子里，就可以看出其中的关联，其中，N和M是两个整数，M是较大的那个数。
现在CXK想要立刻知道这个编码所代表的含义，但是它现在身边没有电脑，你能帮他解决这个问题吗？

输入格式

第一行是两个整数n和m(1<=n,m<=1000)。

接下来一行有一个字符串s，保证长度为n*m,且只由字母组成。

其中数据范围:(1<=n,m<=1000000),(0<=k<=1000000)(1<=x<=n,1<=y<=m)

输出格式

输出n*m的表示法。

样例输入

4 5

ChuTiRenShiZhiZhangA

样例输出

ChuTi
RenSh
iZhiZ
hangA

第四题
涯涯的字符串

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10135

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

涯涯最近在研究算法，他遇到了一个困难的问题，有n个由小写字母组成的字符串。现在要选择尽可能多的字符串，使得每个字母都出现偶数次。
比如有6个字符串 abd,eg,ge,abe,ac,bcd。编号为1,2,3,4,5,6.显然选择第1,2,3,5,6个。

输入格式

第一行是一个数字n(1<=n<=24)。代表接下来有n行字符串。

接下来n行每行有一个只由小写字母组成的字符串。每个字符串的长度不超过10^5。

输出格式

输出最多能选字符串的个数，并按从小到大的顺序输出每个字符串的编号，假设字符串编号从1开始。

样例输入

6
abd
eg
ge
abe
ac
bcd

样例输出

5

1 2 3 5 6

#include<iostream>
#include<cstdio>
#include<cmath>
#include<algorithm>
#include<cstring>
#include<string>
#include<sstream>
#include<stack>

using namespace std;

string s;
long long alpha[30][30];
long long n;
bool pd[30];
long long f[30];
long long maxn,Maxn[30];
long long g;

void dfs()
{
    bool bj=0;
    int fl[30];
    for (int ii=1;ii<=26;ii++)
    {
        fl[ii]=f[ii];
        if ((f[ii]&1)&&f[ii])
            bj=1;
    }
    if (!bj&&g>maxn)
    {
        for (int ii=1;ii<=26;ii++)
        {
            if (pd[ii])
                Maxn[ii]=1;
            else
                Maxn[ii]=0;
        }
        maxn=g;
    }        
    for (int k=1;k<=n;k++)
    {
        if (pd[k])
            continue;
        pd[k]=1;
        for (int ii=1;ii<=26;ii++)
            f[ii]+=alpha[k][ii];
        g++;
        dfs();
        g--;
        for (int ii=1;ii<=26;ii++)
            f[ii]=fl[ii];
        pd[k]=0;
    }
}

int main()
{
    cin>>n;
    for (int i=1;i<=n;i++)
    {
        cin>>s;
        for (int j=0;j!=s.size();j++)
            alpha[i][s[j]-'a'+1]++;
    }
    dfs();
    cout<<maxn<<endl;
    for (int i=1;i<=26;i++)
        if (Maxn[i])
            cout<<i<<' ';
    return 0;
}

正解：DFS+状态压缩
