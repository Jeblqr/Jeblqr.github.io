#OneCode天梯赛 Round2

第一题
横冲直撞的车

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10127

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

有一天无聊的迎哥哥不知道从哪弄来一副象棋，不过不知道下象棋的迎哥哥很是头疼，各种百度查规则ORZ…
这不，我们迎哥哥查阅到，中国象棋是中华民族传统文化的一部分，不仅在国内深受广大棋迷的喜爱，而且流传至国外，也逐步赢得很多棋迷的青睐。国外比较流行国际象棋（chess），很多外国棋友都想知晓中国象棋的游戏规则。两者走棋方法毕竟有很多差异，熟知一下基本规则，两种棋艺很多地方都是能够触类旁通的。很想精通中国象棋的他就开始一个一个的研究象棋中棋子的行棋规则。
迎哥哥第一个研究的就是象棋中的车，车在象棋中威力最大，无论横线、竖线均可行走，只要无子阻拦，步数不受限制。俗称“车行直路”。
那么问题就来了，现在迎哥哥的手上有一个n*m的棋盘，棋盘上已经有k个棋子存在。因为“车行直路”,所以迎哥哥想让他的车想横冲直撞。车能够横冲直撞的条件就是车所在的行或者列不存在别的棋子;已知的k个棋子的位置会干扰车的横冲直撞;所以现在想问你，已知的棋盘有多少个行和多少列列可以让我们迎哥哥的车横冲直撞？

输入格式

第一行输入n,m,k三个整数,棋盘的大小为n*m,k表示棋盘中已有棋子的个数;
接下来k行,每行两个整数x,y表示棋子的位置

其中数据范围:(1<=n,m<=1000000),(0<=k<=1000000)(1<=x<=n,1<=y<=m)

【友情提示】

由于输入数据过大，scanf,printf很有必要哦！！！！

输出格式

输出一行2个整数，分别表示车可以横冲直撞的行数及列数

样例输入

4 2 3
4 1
3 1
4 2

样例输出

2 0

第二题
精英人才

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10128

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

初到杭州的wlxsq,发现杭州搞竞赛的学生是真的多，竞赛的种类也很多。有奥数竞赛，物理竞赛，还有搞OI竞赛的，and so on…是真的很多
现在wlxsq拿到一份杭州竞赛的数据，总共有n个人参加竞赛,假设每人只能参加一种竞赛，每个人有相应的竞赛类型和他的精英价值，现在wlxsq想招一批精英人才，每一类竞赛最多招3人，问你在这个条件下，wlxsq可以招的精英人才的最大价值，当然，对于没有价值的选手(价值不大于0的选手)我们是肯定不会要的。
输入格式
输入一个整数n,表示参加竞赛的总人数，接下来n(1<=n<=10000)行每行输入一个字符串s(由大小写字母表示，长度不超过15)和一个整数v(-100<=v<=100),表示这个竞赛选手的竞赛类型和这次竞赛的精英价值。

输出格式

输出一行，wlxsq可获得的最大精英价值。

样例输入

9
OI 1
OI 2
OI 3
OI 4
OI 5
OI 6
math 10
math 16
english 100

样例输出

141

第三题
统计和

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10129

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

有这样一道有意思的题目，从1到n的自然数中，每次取出不同的两个数，要使他们的和大于m,问你有多少中不同的方案数，取1 5 和 5 1是同一种方案。

输入格式

输入两个整数n,m(1<=n<=10^7,1<=m<=2*10^7)

输出格式

输出一行，和大于m的方案数

样例输入

5 5

样例输出

6

暴力？会爆哎？写了暴力加了优化，怕TLE。归纳了一个多小时的规律，放弃，改回纯暴力吧……8TLE2AC

#include <algorithm>
#include <cmath>
#include <cstdio>
#include <cstring>
#include <iostream>
#include <sstream>
#include <string>
#include<set>
#include<cstdlib>
#include<map>
#define Max 11000
using namespace std;

int n,m;
set<pair<int,int> > pd;
long long ans;

int main()
{
    cin>>n>>m;
    for (int i=n;i>=1;i--)
        for (int j=n;j>=1;j--)
        {
            if (i+j<=m)
                break;
            if (pd.count(make_pair(i,j))||pd.count(make_pair(j,i))||i==j)
                continue;
            ans++;
            pd.insert(make_pair(i,j));
            pd.insert(make_pair(j,i));
        }
    cout<<ans;
    return 0;
}

看了下同学的代码，发现就是枚举好那几个优化。OneCode真是玄学，一段同样的代码和同一个点，可以跑出TLE和72ms?……

#include <algorithm>
#include <cmath>
#include <cstdio>
#include <cstring>
#include <iostream>
#include <sstream>
#include <string>
#include <set>
#include <cstdlib>
#include <map>
using namespace std;

int n,m;
long long ans;

int main()
{
    cin>>n>>m;
    for (int i=n,j=1;i>=1;i--)
    {
        while (i+j<=m&&j<i)//优化一
            j++;
        if (i-j<0)//优化二
            break;
        ans+=i-j;//优化三
    }
    cout<<ans;
    return 0;
}

第四题
字符串拼接

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10130

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

字符串的相关问题一直是一个比较难的问题。这不，一码小学的zbs同学就向吴迎老师问了这样一道题目。给你三个字符串a,b,c,问你c字符串能不能由a,b两个字符串拼接而成，其中a,b字符串中对应的字母顺序不能改变，例如字符串a:zbs,字符串b:wy,字符串c:zwybs,c是能由a,b合成的,如果c为wzsby，则不能有a,b合成。
输入格式
输入一行，3个字符串a,b,c，字符串由小写字母表示，且a,b的长度在[1,1000]区间,c的长度为a,b的长度和.

输出格式

如果c能由a,b合成输出Yes,否则输出No.

样例输入

zbs wy zwybs

样例输出

Yes

没什么好说的，直接暴力DFS，AC。
该题数据水，正解DP

#include <algorithm>
#include <cmath>
#include <cstdio>
#include <cstring>
#include <iostream>
#include <sstream>
#include <string>
#include<set>
#include<cstdlib>
#include<map>
#define Max 11000
using namespace std;

string a,b,c;
int l;

void dfs(string a,string b,string c)
{
    if (a.size()==0&&b.size()==0&&c.size()==0)//如果全部的size都为零，说明可以拼成。直接输出。
    {
        cout<<"Yes";
        exit(0);
    }
    if (a[0]==c[0]&&a.size()!=0)//如果当前a的首字母和c的首字母匹配，DFS下一个。
        dfs(a.substr(1,a.size()),b,c.substr(1,c.size()));
    if (b[0]==c[0]&&b.size()!=0)//如果当前b的首字母和c的首字母匹配，DFS下一个。
        dfs(a,b.substr(1,b.size()),c.substr(1,c.size()));        
}

int main()
{
    cin>>a>>b>>c;
    l=c.size();
    dfs(a,b,c);
    cout<<"No";
    return 0;
}
