第一题
横冲直撞的车

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10127

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

有一天无聊的迎哥哥不知道从哪弄来一副象棋，不过不知道下象棋的迎哥哥很是头疼，各种百度查规则ORZ…
这不，我们迎哥哥查阅到，中国象棋是中华民族传统文化的一部分，不仅在国内深受广大棋迷的喜爱，而且流传至国外，也逐步赢得很多棋迷的青睐。国外比较流行国际象棋（chess），很多外国棋友都想知晓中国象棋的游戏规则。两者走棋方法毕竟有很多差异，熟知一下基本规则，两种棋艺很多地方都是能够触类旁通的。很想精通中国象棋的他就开始一个一个的研究象棋中棋子的行棋规则。
迎哥哥第一个研究的就是象棋中的车，车在象棋中威力最大，无论横线、竖线均可行走，只要无子阻拦，步数不受限制。俗称“车行直路”。
那么问题就来了，现在迎哥哥的手上有一个n*m的棋盘，棋盘上已经有k个棋子存在。因为“车行直路”,所以迎哥哥想让他的车想横冲直撞。车能够横冲直撞的条件就是车所在的行或者列不存在别的棋子;已知的k个棋子的位置会干扰车的横冲直撞;所以现在想问你，已知的棋盘有多少个行和多少列列可以让我们迎哥哥的车横冲直撞？

输入格式

第一行输入n,m,k三个整数,棋盘的大小为n*m,k表示棋盘中已有棋子的个数;
接下来k行,每行两个整数x,y表示棋子的位置

其中数据范围:(1<=n,m<=1000000),(0<=k<=1000000)(1<=x<=n,1<=y<=m)

由于输入数据过大，scanf,printf很有必要哦！！！！

输出一行2个整数，分别表示车可以横冲直撞的行数及列数

样例输入

4 2 3
4 1
3 1
4 2

样例输出

2 0

第四题
字符串拼接

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10130

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

字符串的相关问题一直是一个比较难的问题。这不，一码小学的zbs同学就向吴迎老师问了这样一道题目。给你三个字符串a,b,c,问你c字符串能不能由a,b两个字符串拼接而成，其中a,b字符串中对应的字母顺序不能改变，例如字符串a:zbs,字符串b:wy,字符串c:zwybs,c是能由a,b合成的,如果c为wzsby，则不能有a,b合成。
输入格式
输入一行，3个字符串a,b,c，字符串由小写字母表示，且a,b的长度在[1,1000]区间,c的长度为a,b的长度和.

输出格式

如果c能由a,b合成输出Yes,否则输出No.

样例输入

zbs wy zwybs

样例输出

Yes

没什么好说的，直接暴力DFS，AC。
该题数据水，正解DP


#include <algorithm>
#include <cmath>
#include <cstdio>
#include <cstring>
#include <iostream>
#include <sstream>
#include <string>
#include<set>
#include<cstdlib>
#include<map>
#define Max 11000
using namespace std;

string a,b,c;
int l;

void dfs(string a,string b,string c)
{
    if (a.size()==0&&b.size()==0&&c.size()==0)//如果全部的size都为零，说明可以拼成。直接输出。
    {
        cout<<"Yes";
        exit(0);
    }
    if (a[0]==c[0]&&a.size()!=0)//如果当前a的首字母和c的首字母匹配，DFS下一个。
        dfs(a.substr(1,a.size()),b,c.substr(1,c.size()));
    if (b[0]==c[0]&&b.size()!=0)//如果当前b的首字母和c的首字母匹配，DFS下一个。
        dfs(a,b.substr(1,b.size()),c.substr(1,c.size()));        
}

int main()
{
    cin>>a>>b>>c;
    l=c.size();
    dfs(a,b,c);
    cout<<"No";
    return 0;
}

第二题
贪吃的ZZ恺成

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10133

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

ZZ恺成是个大胃王，有一天，算法“状态规划”发明者，高精度加法不会写者，张焰，买了一大堆零食。她把零食分成了n堆，其中第i堆有i包零食。
也就是说，第1堆有1包零食，第2堆有两包零食，第3堆有3包零食……第n堆有n包零食。然后，由于恺成比较ZZ，每次只会从1堆或者多堆里取走同样多包的零食吃掉。
那么，ZZ恺成至少需要多少次就可以把n堆零食拿完呢？
比如：n=3。那么，一开始零食的状态是1,2,3，先把2,3堆取走2，然后状态变成1,0,1，接着，只要取走1,3堆的1包即可完成。

输入格式

第一行是一个整数t(1<=t<=10^5),代表接下来有t组测试数据。

对于每一组测试数据，只包含一个整数n(1<=n<=10^18)。

输出格式

对于每组测试数据，输出结果，并且占一行。

样例输入

5
1
2
3
4
5

样例输出

1
2
2
3
3

很难看出，这道题有一个不易得的规律：

如果n为奇数，ans(n)=ans((1+n)/2-1）+1;
如果n为偶数，ans(n)=ans((2+n)/2-1）+1;

？？？

咳咳

ans(n)=ans(n/2)+1;

边界条件为:

ans(1)=1;
ans(2)=2;

考虑到该题数据不掺水，采用一丢丢二进制优化以及递归实现，时间复杂度为

O(t*lgn)

第三题
小强的球

原题链接：http://oj.onecode.com.cn/index.php/Ladder/showProblemPage/id/10134

时间限制: 1000 MS 内存限制: 131072 KB

问题描述

小强很喜欢玩球，他想要在桌子上摆一些球，排成一排。
球只有三种颜色，红色，黄色，或者蓝色。现在，小强有R个红球，Y个黄球，B个蓝球，需要依次摆在桌子上。
小强每放一个新的球，可以选择放在球的中间或者放在端头。
每放一个球，小球就可以得到一个分数。计分规则如下：
1.对于刚开始放的第一个球，计0分。
2.如果球放在端头，则分数等于之前放过的球的颜色数，例如之前放球的状态是RYYR，现在把球放在末尾，则计2分。
3.如果球放在某两个球之间，则分数为前半段的球的颜色加上后半段的球的颜色数，例如之前放球的状态是RYYR，现在把球放在两个Y中间，则计4分，因为前半段有两种颜色，后半段也有两种颜色。

那么，小强放球能获取的最大值是多少呢？

输入格式

第一行是一个整数t(1<=t<=10^5),代表有t组测试数据。

接下来t行，每行有三个整数R,Y,B(0<=R,Y,B<=10^9)，分别代表每种颜色球的个数，含义跟题目描述中的一致。

输出格式

对于每组测试数据，输出结果，并且占一行。

样例输入

3
2 2 2
3 3 3
4 4 4

样例输出

15
33
51

#include<iostream>
#include<cstdio>
#include<cmath>
#include<algorithm>
#include<cstring>
#include<string>
#include<sstream>
#include<stack>

using namespace std;

string s;
long long alpha[30][30];
long long n;
bool pd[30];
long long f[30];
long long maxn,Maxn[30];
long long g;

void dfs()
{
    bool bj=0;
    int fl[30];
    for (int ii=1;ii<=26;ii++)
    {
        fl[ii]=f[ii];
        if ((f[ii]&1)&&f[ii])
            bj=1;
    }
    if (!bj&&g>maxn)
    {
        for (int ii=1;ii<=26;ii++)
        {
            if (pd[ii])
                Maxn[ii]=1;
            else
                Maxn[ii]=0;
        }
        maxn=g;
    }        
    for (int k=1;k<=n;k++)
    {
        if (pd[k])
            continue;
        pd[k]=1;
        for (int ii=1;ii<=26;ii++)
            f[ii]+=alpha[k][ii];
        g++;
        dfs();
        g--;
        for (int ii=1;ii<=26;ii++)
            f[ii]=fl[ii];
        pd[k]=0;
    }
}

int main()
{
    cin>>n;
    for (int i=1;i<=n;i++)
    {
        cin>>s;
        for (int j=0;j!=s.size();j++)
            alpha[i][s[j]-'a'+1]++;
    }
    dfs();
    cout<<maxn<<endl;
    for (int i=1;i<=26;i++)
        if (Maxn[i])
            cout<<i<<' ';
    return 0;
}

搭建Hexo博客的心得与搭建教程

===============================

点这里查看我的coding储存库

很久很久之前，就有了自己搭一个博客的想法。不过学业繁忙，一直没有时间。一直到寒假才抽出时间。
稍微了解了下，知道了GitHub，继续百度，了解到了Hexo，继续了解，就是Coding和Markdown。不得不说，Coding很好用，大部分功能又不收费还有在线的ide,业界良心好吧？

然后,开始学习Markdown的语法和博客安装。安装博客的大致方法：

安装git，nmp和Node.js
执行命令：
    sudo npm install -g hexo-cli (安装Hexo)
nmp 
注册一个Coding账号并新建一个库（最好叫 yourname.coding.me  )
新建一个文件夹，储存你的本地博客文件（下面以“blog”做示范）
cd /blog (cd到你的文件夹里面）
hexo init （初始化博客）
打开你的文件夹下的 _config.yml 配置文件，把最后的
deploy:
type: 
改为：
deploy:
  type: git
  repo:
      coding: https://git.coding.net/yourname/yourname.coding.me.git,master
（看好缩进！！！）
保存，退出。
可以先执行命令：hexo s 本地运行，查看效果。
执行命令：npm install hexo-deployer-git --save （安装git插件用于上传blog到Coding）
依次执行：
    hexo clean
    hexo g
    hexo d
没什么意外话就会上传到Coding了，途中可能会让你输入账号和密码，输入就好。
点开“Pages服务”，点开“静态Pages”，在“分支”里面选“master”，点保存。去看看yourname.coding.me是不是有东西了？

配图版有时间会发，敬请期待。
